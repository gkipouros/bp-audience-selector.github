== Description ==

Allow post authors to select who is going to view their posts. This plugin offers Posts Privacy ( Public, Only Me, Friends, Members ).

**Like this plugin? Please consider [leaving a 5-star review](https://wordpress.org/support/plugin/bp-audience-selector/reviews/#new-post).**

**Do you want customizations: Contact me at [gianniskipouros.com](https://gianniskipouros.com/).**

== Installation ==

1. Upload "Audience Selector for BuddyPress" plugin into the directory `wp-content/plugins/`.
2. Enable "Audience Selector for BuddyPress" plugin.
